# FocusPulse

A calm, minimal desktop wellness assistant. Reminds you to drink water, rest your eyes, walk and stretch — without spying on you. **All data stays on your device.**

- 🪟 Glassmorphic Apple/Fluent UI
- ⏱️ Custom reminders (add / edit / delete your own)
- 🎨 Four overlay effects (bubbles, breath, lightning, sunlight)
- 📊 Local analytics, streaks, wellness score
- 🔒 100% local — no account, no cloud, no tracking
- 🖥️ Ships as a real desktop app via Electron (Windows / macOS / Linux)

## Run it on your machine

👉 **See [`RUN.md`](./RUN.md)** for a step-by-step guide to:
- Build the native desktop app (`.exe` / `.app` / Linux binary) with system-tray + auto-launch
- Run it as a Docker service that restarts forever
- Or just run the dev server
